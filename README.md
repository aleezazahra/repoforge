# RepoForge

RepoForge is a high-performance web application designed to automate the creation of professional GitHub documentation. By leveraging the GLM-5 model via the Hugging Face Inference Router, it analyzes repository structures to generate technical README files in seconds.

## Features

* **AI-Driven Documentation**: Utilizes the GLM-5 large language model for technical architectural analysis.
* **Real-time Generation**: Fetches and processes public GitHub repository data through a secure Vercel edge function.
* **One-Click Export**: Integrated clipboard management for immediate deployment of generated markdown.

## Technical Stack

* **Frontend**: React 18, TypeScript, Tailwind CSS
* **Icons**: Lucide React
* **Backend**: Vercel Serverless Functions /NODE JS
* **AI Engine**: GLM-5 (zai-org) via Hugging Face Router v1
